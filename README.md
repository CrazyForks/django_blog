# 特点

- [x] 文章的历史版本管理（随时回退）
- [ ] rss、atom
- [ ] markdown
- [ ] 评论功能
- [ ] 代码高亮
- [ ] 文章页面的toc 导航
- [ ] 新建文章 自动推送到百度、谷歌搜索引擎
